## Analysis
There are two compounding causes:

1. **No static CleanCentive favicon ships.** [frontend/index.html:5](../../../frontend/index.html) references `/vite.svg` (the Vite default), not a CleanCentive icon. There is no `favicon.ico` in `frontend/public/`.
2. **Partner branding overrides the favicon at runtime.** [frontend/src/stores/partnerBrandingStore.ts](../../../frontend/src/stores/partnerBrandingStore.ts) reads a `--partner-favicon-url` CSS variable and replaces the `<link rel="icon">` with the partner's favicon. The partner favicon comes from [backend/src/team/team.service.ts](../../../backend/src/team/team.service.ts) `extractFavicon()`, which scrapes `<link rel="icon">` from the partner's website.

For a user associated with a BaselArea team, the app intentionally swaps in BaselArea's favicon — but the user reports this is undesired. Either the partner-favicon override should not affect the browser tab icon (only in-app branding), or the override should be a smaller composite (CleanCentive icon with partner badge) rather than a full replacement.

The PWA manifest in [frontend/vite.config.ts:9-17](../../../frontend/vite.config.ts) has no `icons` array, so installed PWA also lacks a CleanCentive icon.

## Implementation Plan
1. **Add a real CleanCentive favicon set.**
   - Place `favicon.ico`, `favicon.svg`, `apple-touch-icon.png`, and PWA-sized PNGs (192, 512) in `frontend/public/`.
   - Reference all from [frontend/index.html](../../../frontend/index.html).
   - Add an `icons` array to the VitePWA manifest in [frontend/vite.config.ts](../../../frontend/vite.config.ts).
2. **Stop the runtime favicon swap.** Either:
   - **Option A (recommended):** Remove the favicon mutation from `partnerBrandingStore.ts`. Keep partner branding in-app (header/logo) only — never overwrite the tab icon.
   - **Option B:** Keep the swap but only when the user explicitly opts in, defaulting to CleanCentive's icon.
3. **Verify** with the affected reporter (BaselArea user) and with a non-partner user that the tab now shows CleanCentive's icon.
